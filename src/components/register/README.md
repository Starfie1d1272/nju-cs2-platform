# components/register

报名系统 UI 组件。

## 规划组件

| 组件 | 说明 |
|---|---|
| `RegistrationForm` | 完整报名表单（React Hook Form + Zod，多步骤） |
| `PositionCapBar` | 各位置实时名额进度条（接收 Realtime 推送） |
| `ScreenshotLinkFields` | NJUBox 截图分享链接字段（Rivals 默认 1 个链接，链接内含 5 场截图） |
| `RegistrationSuccess` | 报名成功页组件（提示后续审核与登录入口） |
| `PositionSelect` | 位置下拉选择（满员自动置灰） |
